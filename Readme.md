# Compilation of YAML files for CISCO ENCOR (350-401).

<p>This repository presents a compilation of hands-on study, labs, and notes for the ENCOR (350-401) exam.</p>

## Topics Covered

- OSPFv2 / OSPFv3
- BGP configuration
- STP and RSTP
- VRF configuration
- First hop redundancy (HSRP, VRRP)

  ## Lab environment

  - Cisco Modeling Labs (CML)
  - Cisco IOS images
  - Python scripts for automation
  - Network diagrams
